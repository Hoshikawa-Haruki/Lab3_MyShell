# Open Source Project LAB 3
동의대학교 오픈소스 프로젝트 교과목 LAB 3 과제에 대한 레포지토리입니다.

## 실행 방법
### 1. git clone
``` bash
git clone https://github.com/Hoshikawa-Haruki/Lab3_MyShell.git
```

### 3. 실행파일 실행
```
./<실행파일 이름>
```
